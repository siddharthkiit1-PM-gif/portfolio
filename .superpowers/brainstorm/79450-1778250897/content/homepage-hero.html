Notes on the build (not visible here, but real):

  • Orb = real WebGL shader (R3F + custom GLSL), animated on mouse + scroll, not a CSS gradient.
  • Headline = staggered character reveal, GSAP timeline.
  • Smooth scroll via Lenis; sections snap-pin via GSAP ScrollTrigger.
  • Motion video card = Mux HLS, muted autoplay, lazy-loads off viewport.
  • "Skip to work / Résumé" pill persists in nav forever — recruiter SLA.
  • Reduced-motion media query: orb stills, no parallax, video pauses. Accessibility built-in.
  • Mobile: orb downgrades to static SVG; nav collapses; 60fps target on iPhone 13+.
